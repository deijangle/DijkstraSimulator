# DijkstraSimulator
Simulador do Algoritmo de Dijkstra para grafos, com opção de exibição passo-a-passo. Desenvolvido em C#.

# Requerimentos
- .NET Framework 4.0;
- Biblioteca GraphViz (incluída).

# Como compilar
Para compilar, basta abrir o arquivo Dijkstra.sln (ou Dijkstra/Dijkstra.csproj) no Microsoft Visual Studio e escolher a configuração para compilação (Debug ou Release).

# Exemplos
Na pasta "Exemplos" há um grafo já pronto, contendo a representação de algumas cidades da Romênia e suas distâncias em linha reta.
